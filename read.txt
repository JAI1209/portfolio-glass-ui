# Personal Portfolio Website

A modern, glass-UI inspired personal portfolio website designed to showcase professional skills, projects, certifications, and contact information with a strong focus on visual clarity, responsiveness, and performance.

---

##  Overview

This project is a fully responsive, single-page portfolio website built using **HTML5** and **CSS3**.  
It follows a modular design system with consistent glassmorphism aesthetics, subtle animations, and adaptive layouts across desktop, tablet, and mobile devices.

The website is optimized for:
- Desktop screens
- iPad Mini (portrait & landscape)
- Mobile devices (e.g., Samsung Galaxy S8)

---

## Key Features

- **Glassmorphism UI** with layered blur and depth
- **Responsive layout** using CSS Grid and Flexbox
- **Theme toggle** (dark / accent theme)
- **Animated skill progress bars**
- **Grid-based sections** for Skills, Certifications, and Projects
- **3D animated metallic profile frame**
- **Accessible and readable typography**
- **Optimized contact form layout**
- **No external CSS frameworks**

---

## Tech Stack

- **HTML5** – Semantic markup
- **CSS3** – Custom properties, Grid, Flexbox, animations
- **Font Awesome** – Icons
- **Pure CSS** – No JavaScript frameworks used
